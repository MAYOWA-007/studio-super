# Studio Super brand assets

The full `STUDIO SUPER` editorial wordmark is the primary identity. The interlocking Studio Super ribbon is its compact favicon and app-icon companion.

All creative artwork and creative refinements were produced with the image-generation model. The protected source generations and reviewed transparent masters live in `brand-source/`. Local processing is limited to alpha cleanup, crop, mechanical sizing, and icon plating. No logo artwork is drawn with HTML, CSS, SVG, or application code.

The wordmark has separate generated dark- and light-interface variants. The rounded PNG icon family is intended for web surfaces. `studio-super-mark-ios-1024.png` is the opaque RGB package for native iOS use.
