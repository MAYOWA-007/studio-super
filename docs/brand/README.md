# Studio Super brand assets

The full `STUDIO SUPER` wordmark is the primary identity. The Superscript Tally is its compact favicon and app-icon companion.

All creative artwork and creative refinements were produced with the image generation model. Local processing was limited to alpha cleanup, crop, and mechanical sizing. No logo artwork was drawn with HTML, CSS, SVG, or application code.

The transparent PNG family is intended for web surfaces. `studio-super-mark-ios-1024.png` is the opaque RGB package for native iOS use.
